#   Digital Phone Directory / Smart Contact Book

### 🚀 Overview

Welcome to the official repository of **[Smart Contact Book]**, a robust, GUI-based application. This project was developed as a team effort to provide an intuitive interface for efficient data handling, utilizing advanced data structures for optimal performance.

### ✨ Key Functionalities

* **Intuitive GUI:** Built using Java Swing to provide a seamless and responsive user experience.
* **Efficient Search:** Implemented [Binary/Linear] search algorithms to ensure rapid data retrieval.
* **Data Sorting:** Utilizes [e.g., Merge Sort] to keep records organized systematically.
* **Group Management:** Categorize contacts into custom groups (Family, Work, Friends) for better accessibility.
* **Persistent Storage:** Data is safely stored and backed up, ensuring no information is lost between sessions.

---

### 🛠 Tech Stack & Tools

* **Language:** Java (JDK 21)
* **GUI Framework:** Java Swing, AWT
* **Data Structures & Algorithms:** [e.g., ArrayList, HashMap, Merge Sort, Binary Search]
* **Development Tools:** VS Code, Git/GitHub

---

### 🧩 Project Modules

1. **Core Management Module:** Handles CRUD (Create, Read, Update, Delete) operations.
2. **UI/UX Module:** Manages the modern graphical interface, including dark/light mode switching.
3. **Data Processing Module:** Contains the sorting and searching logic to maintain performance.
4. **Storage/Backup Module:** Manages serialization to save/load contact data.

### 📝 Project Highlights (DSA & Logic)

We focused heavily on performance. By implementing **[e.g., Merge Sort]**, we ensured that even with a large number of contacts, the list remains sorted in $O(n \log n)$ time complexity. Our search implementation uses **[Binary Search]**, significantly reducing lookup time compared to standard list iteration.
